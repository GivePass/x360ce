How To Use - Installation

Run this program from game executable directory. Game executable can look for xinput dll by different names, you can create copy of dll with name provided below till you find one that works:

  • xinput1_3.dll
  • xinput1_2.dll
  • xinput1_1.dll
  • xinput9_1_0.dll

Uninstallation

Delete x360ce.* files, xbox360cemu.ini (if used) and dinput8.dll (if used) from game executable directory.

Note: GTA 4 and Force Feedback

Force feedback works only up to "GTA IV 1.0.3.0" and with "TocaEdit XInput 3.0 Vista" version:
